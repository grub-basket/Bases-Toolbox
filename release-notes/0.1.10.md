# 0.1.10

## Find duplicate notes — big upgrade
- **Merge preview**: pick which note to keep and see exactly what will happen —
  every resulting property tagged by origin (kept, added, merged list, or a
  conflict the kept note wins), which bodies get combined, and what moves to
  trash. No more diffing in your head.
- **Bodies merge in creation-date order** (kept note included), so the combined
  note reads chronologically without you ordering anything. A warning appears
  when creation-date order differs from last-edited order.
- **Clickable note links** — open any candidate in a new tab.
- **Runs as its own tab** (command + launcher Sidebar/Tab/Window buttons), not
  just a dialog; the dialog gains an "Open in a tab" shortcut.
- **No note is pre-selected** as the one to keep — you choose deliberately, and
  the merge button stays disabled until you do.
- **Ignore** groups you don't want to act on: an Ignore button moves a group to
  an "Ignored" tab. Ignores persist and auto-re-flag if the group's membership
  changes. Two tabs — "To review" and "Ignored" — at the top.
- The tab content is centered in a readable column.

## Merges are now fully revertible
- Both the duplicate-finder merge and "Merge current note into another" are
  recorded to **History** as whole-note snapshots. Reverting restores the kept
  note, **recreates the trashed sources**, and undoes backlink rewrites in
  other notes — the entire merge, in one click.
- The merge/revert confirmations now show the result **in the button** (it turns
  green on success) and use clearer wording.

## Launcher
- Each panel can now open in a **new window** (Sidebar / Tab / Window).
- New **wrench** icon (the old one looked like Canvas/Workspaces).

## Clearer History
- "Force-revert drifted files" is relabeled and explained: reverting normally
  skips notes you edited again after the change (so it never overwrites newer
  work); this option reverts them too. Merge reverts always restore the whole
  note and ignore this setting.
