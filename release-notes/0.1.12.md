# 0.1.12

## CSV import & export — separate, first-class panels
- **Import** and **Export** are each their own feature now, openable in the
  **sidebar, a tab, or a new window** (or a dialog).
- **Export → From a base**: pick any `.base` file (no need to open it). It reads
  the folders the base references and its column order, resolves the built-in
  `file.*` columns (name, path, folder, ext, size, created, modified, tags) plus
  your frontmatter properties, and exports the notes it covers. Multi-folder
  bases are handled; a base with filters beyond folder scope is flagged as a
  best-effort superset.
- **Export → From a folder**: pick any folder — every note's frontmatter becomes
  a CSV row with a column per key found. Preview, then **Copy for Excel** or
  write a `.csv`.
- **Export active base results** (command): still available for a base's exact
  live filtered results with formula columns; errors on a non-base view.
- **Import** now autocompletes the target folder and rejects spreadsheet files
  (`.xlsx`/`.numbers`/…) with a "convert to CSV first" note instead of reading
  binary as garbage.

## Bases built-in properties
- The **property index** has a new collapsible **"Bases built-in properties"**
  section: click any `file.*` identifier (created time, tags, folder, …) to copy
  it, then paste it into a base when its property menu "forgets" it. They're
  identifiers, not formulas.
- The settings **Reference** and the README document the same list, with
  `.base` YAML examples and when to use a formula instead.

## Reverts are clearer & safer
- Each history entry shows a **red data-loss warning on its own line** when
  reverting it could lose data — always for merges, and for other operations
  only when "force" is on.
- After a revert, the entry lists **which files were skipped and why**
  (edited since / renamed / missing), each clickable.
- Long paths and change values now wrap instead of colliding.

## Tips (README)
- Fork a note with the Commander plugin (Create new + Rename).
- Use Obsidian's built-in formula editor or Formula Forge for formulas.
