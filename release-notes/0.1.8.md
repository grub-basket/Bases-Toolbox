# 0.1.8

## Launcher
- A friendly **Bases Toolbox launcher** in the left sidebar (new ribbon icon +
  "Open Bases Toolbox launcher") lists every feature as a card with a one-line
  description and a button to open it — grouped into **Panels & windows** (each
  with a Sidebar and a Tab button), **Dialogs & actions**, and **Settings**.
  Great for people who don't want to hunt the command palette.

## Conditional formatting
- New **"is duplicated (in this base)"** rule — colors any row or cell whose
  value appears more than once among the base's currently-shown rows (spot
  duplicate account names, IDs, etc. at a glance).

## Find & replace
- Cleaner layout: each control on its own row (Property, Find, then Replace
  with + the text box).
- **"Empty replacement clears the value"** now shows as a red warning with an
  icon when the box is empty (it's a destructive clear), and tempers to a plain
  note once you type a value.
- The "N files will change" summary is bigger and sits right above the list.
- The command is now **"Find and replace property values"** so typing "find and
  replace" finds it in the palette.

## Property format doctor
- Each property group heading shows its **type icon** (matching the property
  index).

## Fixes
- **Rollup / bulk edit** no longer say "open a base" when a base *is* open —
  they now find the open base even when it isn't the focused view (e.g. launched
  from the launcher or a sidebar).

## Settings
- The in-app **reference** now explains every feature in plain language with
  examples (rollup, forks, companions, allowed values, and more), so
  non-technical users can learn what each tool does without leaving Obsidian.
