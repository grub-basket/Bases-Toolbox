# 0.1.7

## Views — move between main tab and sidebar
- Every plugin view (Property Index, Find & Replace, History, Format Doctor,
  Conditional Formatting) gains a **"Move to sidebar"** action in its tab header
  — dock a main-tab view into the right sidebar. The main-tab-primary views also
  get the reverse **"Open in a main tab"**, so you can move them either way.

## Property format doctor
- **Ignore** a mismatch you're fine leaving as-is. It's hidden from the list but
  keyed to the value — so if that value ever **changes ("breaks"), it re-flags
  automatically**. No permanent ignore that silently goes stale.
- New **"To fix" and "Ignored" tabs** at the top. The ignored values live on
  their own tab, each with an **Un-ignore** button, plus **Clear ignore list**.
