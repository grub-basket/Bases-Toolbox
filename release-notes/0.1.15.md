# 0.1.15

Hardening from a full-codebase audit — no user-facing feature changes.

- **Reserved property names are rejected** across all property-writing tools
  (bulk edit, the properties modal, rollup, metadata stamp): a name like
  `__proto__` used to silently succeed while writing nothing. It's now refused
  with a clear notice.
- **Base export honors `not(...)` filters.** A base filtering *out* a folder
  (`not(file.inFolder("Archive"))`) exported *only* that folder; it now scopes
  correctly (whole-vault superset, flagged as approximate).
- **Merge revert survives a deleted folder.** If a merged note's folder was
  deleted afterward, reverting the merge now recreates the folder and both
  notes instead of aborting halfway.
