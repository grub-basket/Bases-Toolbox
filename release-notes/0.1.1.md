# 0.1.1 — store-review fixes

Addresses the community-store review findings. No feature changes.

- Replaces all direct inline-style assignments with Obsidian's `setCssStyles`
  (conditional-formatting row tints, embed height flags, the allowed-values
  picker position, and show/hide of conditional settings rows) — the review's
  one blocking error.
- Popout-window compatibility: DOM listeners, queries, and body classes use
  `activeDocument`; timers use `window.setTimeout`; element type checks use
  Obsidian's cross-window `instanceOf`.
- Note merging now uses `FileManager.trashFile()` so deletions respect the
  user's configured trash preference (system trash / vault trash / permanent).
- Replaces deprecated `Workspace.activeLeaf` with `getActiveViewOfType` in
  bulk edit and CSV export.
- Number guard: allows the comma decimal separator explicitly and drops an
  unnecessary regex escape.
- Async click handlers wrapped to satisfy void-return expectations; removes
  unused code (`ALL_VALUES` sentinel, an unused type import); fixes a
  redundant `unknown | null` union; removes the one `!important` from the
  stylesheet in favor of a more specific selector.
