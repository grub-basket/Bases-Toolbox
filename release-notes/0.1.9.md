# 0.1.9

## Find duplicate notes
- Note entries are now **clickable links** — click one to open that note in a
  new tab instead of squinting at a plain path.
- The finder can run as **its own tab** (not just a dialog): a new command
  **"Find duplicate notes (in a tab)"**, plus **Sidebar / Tab** buttons in the
  launcher. In a tab, opened notes sit beside the finder instead of behind it.
  The dialog gains an **"Open in a tab"** shortcut.
- **No note is pre-selected** as the one to keep — you pick it deliberately,
  and the merge button stays disabled until you do (changing your pick also
  re-arms the confirm step).
- Tab content is centered in a readable column instead of stretching across
  the whole window.

## Docs
- The README now has a **Permissions & data** section: no network access or
  telemetry, vault file listing is local-only, and clipboard use is write-only.
